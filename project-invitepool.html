InvitePool

React • JavaScript • CSS • Supabase

Problem

Event planning can become disorganized when invitations, RSVP tracking, and guest communication are handled across multiple places. I wanted to build a cleaner and more user-friendly way for hosts to create event pages, share event details, and manage guest responses all in one application.

Approach

I built InvitePool as a full-stack web application using React for the front end and Supabase for backend services and data storage. The project focused on making event creation simple while giving hosts useful tools to manage invitations, RSVP activity, and event details.

Solution

InvitePool includes:

  • Custom event creation with live preview
  • Public and private invite pages
  • RSVP form with guest response tracking
  • Duplicate RSVP handling workflow
  • Host dashboard for event and RSVP management
  • Responsive design for a cleaner user experience

Outcome

This project helped me strengthen my front-end development skills, work with a backend service, and think more deeply about user experience, workflow design, and practical application building.
